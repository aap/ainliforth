# Forth in PDP-11 machine code

(Ainlif is gothic for eleven)

## How to run
 - start pdp11
 - load papertape with "l[oad] pt"
 - run main with "g[o]"

## What's implemented

The dictionary contains atm:
- foo (constant 1234)
- .
- .s
- +
- -
- *
- /
- mod
- dup
- exit
- (literal)
- [
- ]
- :
- ;
- +1 (adds one to TOS)
- 1sq (adds one to TOS and squares)
